# 🧠 FaceNetHub – Intelligent Face Recognition Web System

FaceNetHub is a complete **AI-powered face recognition web application** that allows users to **recognize, verify, search, multiface detection and register individuals using facial data**.
It combines **Machine Learning + Web Development** to simulate a real-world biometric system.

---

## 🚀 Features

### 🔍 Face Recognition 

* Upload image or capture via webcam
* Identifies person from database
* Displays full user details

### 🔁 Face Verification 

* Compare two faces
* Determines if both belong to the same person

### 👤 Person Search

* Search using:

  * ID
  * Email
  * Mobile number
* Displays stored details and image

### 📷 Search by Face

* Upload or capture image
* Finds matching person in database

### 👤 Multiface Detection

* Upload a group image
* recognizes the images if in database.

### ➕ Register New Person

* Add new users using:

  * Image upload
  * Real-time camera capture
* Automatically saves:

  * Image → dataset
  * Data → database

* **Machine Learning:** TensorFlow / ANN
